# ConsentShield PHP SDK

Official PHP 8.1+ client + Laravel/Symfony integration for the [ConsentShield Compliance API](https://api.consentshield.in/v1).

- **Packagist**: `consentshield/php` 1.0.0 (compliance-contract wrapper)
- **Packagist**: `consentshield/consentshield` 1.0.0 (raw client; non-Guzzle callers)
- **Source**: <https://github.com/consentshield-org/php>

## Install

```bash
composer require consentshield/php:^1.0
```

```php
use ConsentShield\Sdk\ConsentShieldClient;

$client = ConsentShieldClient::create(getenv('CONSENTSHIELD_API_KEY'));
$ping = $client->utility()->ping();
```

## Compliance contract

| Outcome | Default behaviour |
|---|---|
| 4xx response | `ConsentShieldApiException` — ALWAYS surfaces, never retried |
| Per-attempt timeout | `ConsentShieldTimeoutException` — NEVER retried |
| 5xx + transport failure (fail-CLOSED, default) | `ConsentVerifyException` with `VerifyFailureCause` |
| 5xx + transport failure (fail-OPEN) | open outcome with cause set |

Retry budget: 3 retries on 5xx + transport with 100 / 400 / 1600 ms exponential backoff. Per-attempt timeout 2 s. Configurable via the options array passed to `ConsentShieldClient::create()`.

## Source layout

- `generated/` — generated by `openapi-generator-cli@v7.10.0` from `app/public/openapi.yaml` in the upstream monorepo. Do NOT hand-edit.
- `wrapper/src/` — `ConsentShieldClient` factory + `RetryMiddleware` (Guzzle middleware) + exception hierarchy.
- `wrapper/tests/` — PHPUnit test suite.
- `examples/laravel-middleware/` + `examples/symfony-controller/` — runnable framework examples.
